## Lisbon Design Meetings

Quick-start instructions:

```bash
~ git clone git@github.com:asirgado/EDLxLDM.git EDLxLDM
~ cd /EDLxLDM
~ $ bower install & npm install
~ $ grunt serve
# => Now browse to http://localhost:9000
```
